Scala with some comparisons to Java

The Basics

Package

Java


package myPackage ;

Scala


package myPackage

Folder structure

  • Java:

    • Folder structure matches package structure

  • Scala:

    • Any folder structure. Unrelated to package structure

    Any file can be in any package regardless of folder

Imports

Java


import java.util.Map  ;
import java.util.List ;
import java.io.*      ;

Scala


import java.util.map
import java.util.List
import java.io._
import java.lang.reflect.{Member,Field,Method}

Scala imports can occur inside a function/closure too!

Immutable variables

Java


final String name = "Shusaku" ;

Scala


val name : String = "Shusaku"

Note: Scala defaults to public scope

Mutable variables

Java


String name = "Shusaku" ;
name = "Go Seigen" ;

Scala


var name : String = "Shusaku"
name = "Go Seigen"

or


var name = "Shusaku"

Java Bean



package myPackage ;

import myPackage.mySubPackage ;
import java.util.Objects ;

class Person {

  private String givenName  = null ;
  private String familyName = null ;

  public Person( givenName : String , familyName : String ) {
    this.givenName = givenName ;
    this.familyName = familyName ;
  }

  public String getGivenName() {
    return this.givenName ;
  }

  public String getFamilyName() {
    return this.familyName ;
  }

  @Override
  public String toString() {
    return this.givenName + " " + this.familyName ;
  }

  @Override
  public int hashCode() {
    return Objects.hashCode(this.firstName, this.lastName);
  }

  @Override
  public boolean equals( Object aThat ) {
    boolean isEqual = true ;
    if( aThat == null ) {
      isEquals = false ;
    } else {
      if ( this != aThat ) {
        if ( !(aThat instanceof Person) ) {
          isEquals = false ;
        } else {
          Person that = (Person)aThat;
          if( that.hashCode() != hashCode() ) {
            isEquals = false ;
          }
        }
      }
    }
    return isEqual
  }

}

Scala Case Class


package myPackage

import myPackage.mySubPackage

case class Person ( val givenName : String
                  , val familyName : String
                  ) {
  def toString() : String = {
    s"${givenName} ${familyName}"
  }
}

Structure

Traits


package myPackage

import myPackage.mySubPackage

trait Animal {
  def getLegCount() : Int
}

class Dog extends Animal {
  override def getLegCount() : Int = {
    4
  }
}

Note: It's OK having traits and classes in the same source file

Chained Constructors


package myPackage

import myPackage.mySubPackage

abstract class Pet( license : LicenseID ) {
  ...
}

class Dog( license : LicenseID ) extends Animal( license ) {
  ...
}

Note: It's OK having two classes in the same source file

Companion Objects


class MyClass {
  println( MY_CONSTANT )
}

object MyClass {
  val MY_CONSTANT = "hello world"
}

Objects can implement interfaces


trait NamedThing {
  def getName() : String
}

object MyClass extends NamedThing {
  override def getName() : String = {
    "hello world"
  }
}

Invoking code through main


object MyObject {
  def main( args : String* ) : Unit = {
    println( "hello world" )
  }
}
